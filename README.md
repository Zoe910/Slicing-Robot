# Slicing-Robot

## Introduction
This project is a real-time video deconstruction experiment we built using p5.js and ml5.js. We used FaceMesh to isolate the face area and apply a horizontal "slicing and stretching" effect, making it look like a glitchy, broken TV signal. On top of that, we added Handpose tracking so our fingertips can trail purple particles, making us feel like cyber-robots literally tearing through digital space-time.
